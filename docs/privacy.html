事件倒计时提醒 - 隐私政策

感谢您使用事件倒计时提醒扩展。本隐私政策旨在说明我们如何收集、使用和保护您的个人信息。

信息收集

我们只收集以下必要信息：

  • 您创建的事件数据（事件名称和日期）
  • 基本的使用统计信息

信息存储

所有数据都存储在您的本地浏览器中，我们不会将任何数据上传到服务器。具体包括：

  • 事件数据存储在浏览器的本地存储中
  • 不会收集或存储任何个人身份信息

信息使用

收集的信息仅用于：

  • 显示事件倒计时
  • 发送提醒通知
  • 改进扩展功能

第三方服务

本扩展不使用任何第三方服务，也不会与第三方共享您的信息。

数据安全

我们采取适当的安全措施来保护您的数据：

  • 所有数据仅保存在本地
  • 不进行网络传输
  • 遵循浏览器扩展的安全最佳实践

您的权利

您可以随时：

  • 查看、修改或删除您的事件数据
  • 卸载扩展，这将删除所有本地存储的数据

政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会通过扩展更新通知您。

联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

  • GitHub Issues: https://github.com/gitLopp/countdown-reminder/issues
