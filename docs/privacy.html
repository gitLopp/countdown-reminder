事件倒计时提醒 - 隐私政策

感谢您使用事件倒计时提醒扩展。本隐私政策说明我们如何收集、使用和保护您的个人信息。

信息收集

我们只收集以下必要信息：

  • 您创建的事件数据（事件名称和日期）
  • 基本使用统计信息

信息存储

我们使用浏览器提供的同步存储功能来保存您的数据：

  • 事件数据存储在浏览器的同步存储中，可在您登录的不同设备间自动同步
  • 同步功能仅在您登录浏览器账户时可用
  • 我们不会将数据上传到任何第三方服务器
  • 不收集任何个人身份信息

信息使用

收集的信息仅用于：

  • 显示事件倒计时
  • 发送提醒通知
  • 在您的设备间同步事件数据
  • 改进扩展功能

第三方服务

本扩展不使用任何第三方服务，也不会与第三方共享您的信息。

数据安全

我们采取适当的安全措施来保护您的数据：

  • 所有数据仅保存在浏览器的同步存储中
  • 数据传输仅限于浏览器自身的同步功能
  • 遵循浏览器扩展安全最佳实践

您的权利

您可以随时：

  • 查看、修改或删除您的事件数据
  • 通过浏览器的扩展管理页面禁用数据同步
  • 卸载扩展，这将删除所有本地存储的数据

政策更新

我们可能会不时更新本隐私政策。对于重大变更，我们将通过扩展更新时通知您。

联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

  • GitHub Issues：https://github.com/gitLopp/countdown-reminder/issues
