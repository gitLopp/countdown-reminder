Event Countdown Reminder - Privacy Policy

Thank you for using the Event Countdown Reminder extension. This privacy policy explains how we collect, use, and protect your personal information.

Information Collection

We only collect the following necessary information:

  • Event data you create (event names and dates)
  • Basic usage statistics

Information Storage

All data is stored locally in your browser, and we do not upload any data to servers. Specifically:

  • Event data is stored in the browser's local storage
  • No personal identification information is collected or stored

Information Usage

The collected information is only used for:

  • Displaying event countdowns
  • Sending reminder notifications
  • Improving extension functionality

Third-Party Services

This extension does not use any third-party services and does not share your information with third parties.

Data Security

We take appropriate security measures to protect your data:

  • All data is saved locally only
  • No network transmission occurs
  • Following browser extension security best practices

Your Rights

You can at any time:

  • View, modify, or delete your event data
  • Uninstall the extension, which will remove all locally stored data

Policy Updates

We may update this privacy policy from time to time. For significant changes, we will notify you through extension updates.

Contact Us

If you have any questions about this privacy policy, please contact us through:

  • GitHub Issues: https://github.com/gitLopp/countdown-reminder/issues
