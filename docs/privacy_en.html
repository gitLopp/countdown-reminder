Privacy Policy

Last updated: March 2024

Data Collection and Usage

The Event Countdown Reminder extension values your privacy. We only store the following information locally:

  • Event names you create
  • Event dates
  • Event IDs (for internal management)

Data Storage

All data is stored only in your browser's local storage and is never uploaded to any servers or shared with third parties. You can clear this data at any time through your browser's extension management interface.

Permission Usage

This extension uses the following permissions:

  • storage - Used to save your event data locally
  • alarms - Used to periodically check events and update countdowns
  • notifications - Used to send reminders when events are approaching
  • action - Used to display the extension icon and badge

Data Deletion

You can delete your data in the following ways:

  • Delete individual events through the extension interface
  • Clear extension data through your browser's extension management page
  • Uninstall the extension to completely remove all related data

Updates

We may update this privacy policy from time to time. Any significant changes will be notified to you when updating the extension.
